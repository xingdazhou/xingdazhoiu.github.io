    # 网络编程

    状态码：状态码由三个数字组成，第一个数字定义了响应的类别

    1**：指示信息--表示请求已接收，继续处理。

    2**：成功--表示请求已经成功接收

    3**：重定向--要完成请求必须进一步操作

    4**：客户端错误--请求有语法错误或请求无法实现，页面不存在等

    5**：服务端错误--服务器未能实现合法的请求

    # 常见的状态码

    200：ok 客户端请求成功

    400：Bad Request 客户端请求有语法错误，不能被服务器所理解

    401：请求未经授权

    403：服务器收到请求，但是拒绝提供服务。

    404：请求的资源不存在，比如输入了错误的url，

    500：服服务器发生了不可预期的错误

    503：服务器当前不能处理客户端请求，一段时间后可能恢复正常
