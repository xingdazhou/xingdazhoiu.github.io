    # 命名视图

    一个组件内部有多个router-view该如何分配组件？比如三栏布局，顶部栏点击按钮，左侧栏的菜单变化

    这个命名视图就是给视图添加一个name属性。比如说在App.vue中有两个router-view，一个具名视图，一个匿名视图

    # 异步组件

    其实是一个要导入函数

    component : import ('./components/PageA')

    这样写的好处就是按需加载。就是只有进入这个组件后，这个组件才会被加载过来。一开始并不会加载这个组件。

    也叫懒加载。
