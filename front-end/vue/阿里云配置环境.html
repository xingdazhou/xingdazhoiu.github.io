    # （npm run dev）.

    
注意，下面两种方式是等效的：

1. npm run dev

2. pm2 start  npm -- run dev

语法：pm2 start npm --watch --name <taskname> -- run <scriptname>;

其中 -- watch监听代码变化，-- name重命名任务名称，-- run后面跟脚本名字

至此，PM2实现npm run 来启动 node 的方法已经实现。
