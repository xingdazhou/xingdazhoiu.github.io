    # react

    import导入和export导出对应有三种方式

    // 导入方式三种
import test1 from './test1.js'
import {num,age} from './test2.js'
import * as obj from './test3.js'

//导出方式三种
//test1.js
export default 'zhouxingda'

//test2.js
export let num = 2
let age = 18
export {age}

//test3.js 
export let num = 1
let age = 2
export {age}
export default "你好"

    # import & require

    • import是es6的导入，require是common.js的导入

    • import实现在浏览器，require实现在node

    • import属于加载前置的机制，因此将其全部放在代码顶部，代码解析逐个获取一个引入列表，先引入依赖，在向下执行代码 加载前置

    • require

      • 可以这样用if(true){let querystring = require("querystring")}
      • 加载滞后，代码执行到哪一行，才进行加载
